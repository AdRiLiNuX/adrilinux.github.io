Déploiement de Traefik 2.0 sur docker swarm

Traefik 2.0 est sortie en version finale il y a quelques jours et j’ai donc effectué la migration de ma configuration. Je savais que la configuration avait subie de gros changement, mais j’ai quand même rencontrer quelques difficultées sur les points avancés. Un trucs que je n’avais pas compris tout de suite, c’est qu’on peut utiliser les middleware à différents endroits, ce qui permet de ne pas dupliquer du code pour chaque service déployer.
